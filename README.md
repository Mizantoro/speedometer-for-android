# Speedometer for Android
I needed a simple app for measuring my speed without looking at the screen while skiing. None of the existing apps satisfied me, so with help of Stack Overflow and ChatGPT I did the most reasonable thing: I did it myself. Since I am new to kotlin and Android programming this app is provided as-is and I do not guarantee that the app is safe to use.
